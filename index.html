Research Projects

Curved Plate and Frame Heat Exchanger

January 2022

This project was an attempt to improve the efficiency of the plate heat exchanger using curved plates instead of traditional flat plates. SOLIDWORKS, HTRI Xchanger and ANSYS FLUENT were used extensively for design and simulation purposes.

Lungs Performance Analysis Using Spirometry

July 2021

In the post-Covid situation, this project was an attempt to design an efficient yet affordable device that can predict the lung condition of the user. This was achieved by measuring the forced vital capacity of the user with an Arduino and a machine learning model trained with MATLAB.

Mars Rover - Team Interplaneter (BUET)
